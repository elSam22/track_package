# Track Package

Suivi de colis via scraping de parcelsapp.com avec Selenium et BeautifulSoup.

## Installation

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```


## Utilisation

Suivi en ligne de commande

`python src/tracker.py`

### Interface web

`python src/app.py`
